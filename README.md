# 🌍 Smart Travel Planner Web App

A full-featured web application designed to simplify and enhance travel planning by providing **real-time suggestions**, **collaborative itinerary editing**, and **intelligent route optimization**.

---

## 🚀 Key Features

- 🧠 **Smart Itinerary Builder**
  - Real-time suggestions based on location, preferences, and travel history
  - Dynamic travel log creation and editing

- 🗺️ **Google Maps Integration**
  - Live map previews of selected destinations
  - Route optimization using Google Maps API and location-based data

- 🤝 **Collaborative Planning**
  - Multi-user editing of travel plans
  - Shared itineraries with roles (editor/viewer)

- 🔐 **Secure Authentication**
  - OAuth + Passport.js-based user login
  - Session management and role-based access control

## 🛠️ Tech Stack

| Technology     | Role                                 |
|----------------|--------------------------------------|
| Node.js        | Backend server logic                 |
| Express.js     | RESTful API and server framework     |
| MongoDB        | Database for users and itineraries   |
| Passport.js    | Secure authentication and sessions   |
| OAuth 2.0      | Third-party login support            |
| Google Maps API| Route planning and location services |
